# Cyber Security Beginner Hub

A platform under development dedicated to helping beginners navigate their journey into the world of Cyber Security. 

## Vision
To provide a curated roadmap, essential learning resources, and a supportive environment for individuals taking their first steps in digital security.

## Planned Features
- **Learning Roadmap:** A structured guide for beginners.
- **Resource Library:** Links to trusted tutorials and practice environments.
- **Tools & Tips:** Practical advice for security enthusiasts.

## Tech Stack
- Framework: Next.js
- Styling: Tailwind CSS
